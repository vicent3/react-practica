Que es babel
    nos permite usar caracteristicas nuevas de js
    En el desarrollo de software, Babel es una herramienta popular
    que se utiliza principalmente para transpilar código 
    JavaScript. Su función principal es convertir código escrito
    en versiones más nuevas de JavaScript 
    (como ECMAScript 6+ o versiones posteriores) 
    a versiones más antiguas, para asegurar la compatibilidad 
    con navegadores que no soportan las características más 
    recientes del lenguaje. Esto permite a los desarrolladores 
    escribir código moderno sin preocuparse por la compatibilidad
    con todos los navegadores.

    Babel también se usa para convertir JSX 
    (una sintaxis especial de JavaScript usada en React) 
    en JavaScript estándar, facilitando el desarrollo de 
    aplicaciones web.
